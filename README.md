#Netflix Clone (HTML & CSS)
A visually appealing clone of the Netflix app created using only HTML and CSS. This project replicates the core design elements of Netflix’s user interface including layout, colors, typography, and responsive design.

#Features
Clean and modern UI resembling Netflix's homepage.
Responsive layout that adapts across desktops and mobile devices.
Hover effects for movie thumbnails to enhance interactivity.
Static showcase of movie and TV show titles (no video playback functionality).
Pure front-end implementation without JavaScript.

#Installation

git clone https://github.com/yourusername/netflix-clone-html-css.git

#Limitations

This is a static front-end project. It does not include interactive video playing, user login, or dynamic content loading.
The app uses placeholders and static content to mimic Netflix’s look and layout.
